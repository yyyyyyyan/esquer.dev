Removedor de metadados

O removedor de metadados do esquer.dev nada mais é do que uma interface para a ferramenta mat2 - um programa de linha de comando para remoção de metadados de diferentes tipos de arquivos, escrito em Python 3.

Como nem todos podem nem querem instalar o programa em seus computadores, a ferramenta no esquer.dev executa o mat2 dentro de nosso servidor. Isso significa que seu arquivo passa dentro de nosso servidor por um instante, enquanto fazemos a remoção de metadados, mas não se preocupe - nenhum dado é lido ou armazenado por nós. Só fazemos a limpeza e depois te mandamos de volta!

O removedor de metadados foi apresentado na postagem Como protestar com segurança (digital).
